# VS Code Shortcuts - KitchenHelper

## Claude Code
- `Ctrl + Shift + C` - Claude Code starten (im Workspace)
- `Ctrl + Shift + F` - Aktuelle Datei fixen

## Backend
- `Ctrl + Shift + B` - Backend Server starten

## VS Code Standard
- `Ctrl + `` - Terminal öffnen/schließen
- `Ctrl + Shift + `` - Neues Terminal
- `Ctrl + P` - Datei suchen
- `Ctrl + Shift + P` - Command Palette
- `F5` - Debugging starten

## Terminal Navigation
- `Ctrl + PageUp/PageDown` - Terminal wechseln
- `Alt + ↑/↓` - Zeile verschieben
- `Ctrl + /` - Zeile kommentieren
